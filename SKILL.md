# Stock Deep Analyst — 个股深度分析与投资策略 Skill

**用途**：对 A 股个股进行专业级深度分析，基于最新业绩公告（预告/快报/年报/季报）生成完整的投资分析报告与操作策略。

**前置依赖**：需要 `westock-data` skill 已安装（用于获取行情、财务、技术指标等原始数据）。

---

## 分析触发条件

当用户提出以下类型的请求时，加载本 Skill：

- "分析一下 XX 的业绩预告/快报/年报"
- "XX 最新业绩出了，帮我分析一下"
- "给我出一份 XX 的投资策略"
- "XX 值不值得买？"
- "帮我深度研究一下 XX"

## 标准分析流程（SOP）

### Phase 1：数据采集

**必须采集的数据清单（按顺序执行）：**

```
# 1. 搜索确认股票代码
westock-data search <股票名称>

# 2. 实时行情（价格/PE/PB/PS/市值/换手率/52周高低等）
westock-data quote <code>

# 3. 最新公告列表（找到业绩预告/快报/年报公告）
westock-data notice <code>

# 4. 公告原文（获取业绩预告/快报的完整文本）
westock-data ncontent <公告ID>

# 5. 财务报表（最近8期利润表，用于季度拆解）
westock-data finance <code> lrb 8

# 6. 技术指标（全部指标）
westock-data technical <code> all

# 7. 筹码成本分析
westock-data chip <code>

# 8. 资金流向
westock-data asfund <code>

# 9. 机构一致预期
westock-data consensus <code>

# 10. 机构评级与最新研报
westock-data rating <code>

# 11. 最新研报列表（取前5条）
westock-data report <code> 1 5 1
```

**可选数据（视情况采集）：**

```
# 股东结构变动
westock-data shareholder <code>

# 分红数据
westock-data dividend <code>

# 龙虎榜（如有涨停/跌停）
westock-data lhb <code>

# K线数据（如需画趋势）
westock-data kline <code> day 60 qfq
```

### Phase 2：分析框架

报告**必须包含以下 10 个板块**，缺一不可：

---

#### 板块一：业绩公告核心数据

- 提取公告原文中的所有数值数据
- 制作核心指标对比表（本期 vs 上期 vs 同比增速）
- 必须包含：营收、营业利润、归母净利、扣非净利、EPS、ROE
- 如有资产负债数据：总资产、净资产、每股净资产

**格式要求：**
```markdown
| 核心指标 | 本报告期 | 上年同期 | 同比增长 |
|---------|---------|---------|---------|
| 营业总收入 | xxx亿元 | xxx亿元 | +xx.xx% |
| ... | ... | ... | ... |
```

#### 板块二：季度拆解与趋势分析

- 用8期利润表数据拆解出**单季度**营收和净利润
- 计算环比增速
- 判断是否逐季增长、是否有季节性规律
- 如果是Q1预告，特别关注是否打破"淡季规律"

**关键判断逻辑：**
- 环比持续增长 → 趋势向好
- Q1环比Q4增长 → 超季节性（重大利好信号）
- 环比放缓但仍增长 → 正常的高基数效应
- 环比下降 → 需要警惕

#### 板块三：超预期分析

- 将实际/预告数据与**机构一致预期**对比
- 判断是否超预期、符合预期、低于预期
- 计算预告增速 vs 机构预期增速的差异
- 与行业可比公司增速对比

**超预期判断标准：**
- 实际增速 > 一致预期增速 + 5pct → 明显超预期
- 实际增速 ≈ 一致预期增速 ±5pct → 符合预期
- 实际增速 < 一致预期增速 - 5pct → 低于预期

#### 板块四：估值分析

**必须计算的估值指标：**

| 指标 | 计算方式 | 用途 |
|------|---------|------|
| PE (TTM) | 从 quote 获取 | 当前估值水平 |
| PE (Forward) | 市值 / 机构预期净利 | 前瞻估值 |
| PB | 从 quote 获取 | 资产端估值 |
| PS | 从 quote 获取 | 营收端估值 |
| PEG | PE / 净利增速 | 成长性价比 |
| 股息率 TTM | 从 quote 获取 | 分红回报 |

**PEG 评判标准（A股成长股）：**
- PEG < 0.5 → 极具吸引力
- 0.5 ≤ PEG < 0.8 → 较有吸引力
- 0.8 ≤ PEG < 1.0 → 合理
- 1.0 ≤ PEG < 1.5 → 偏贵
- PEG ≥ 1.5 → 高估

**合理价值计算（至少用3种方法）：**
1. PE估值法：预期净利 × 合理PE倍数
2. PEG=1法：增速 × EPS
3. 机构一致目标价

#### 板块五：技术面分析

**必须分析的指标：**

- **均线系统**：MA5/10/20/30/60/120/250，判断多头/空头排列
- **MACD**：DIF vs DEA，金叉/死叉，红柱/绿柱趋势
- **KDJ**：J值超买(>80)/超卖(<20)判断
- **RSI**：RSI(6)/RSI(12)/RSI(24)，超买(>70)/超卖(<30)
- **BOLL**：现价与上轨/中轨/下轨的关系
- **CCI**：超买(>100)/超卖(<-100)

**综合判断模板：**
- 均线全线多头 + MACD金叉 + RSI<70 → **强势且未超买，可积极做多**
- 均线全线多头 + RSI>80 → **趋势强但短期超买，不宜追高**
- 均线空头排列 + MACD死叉 → **弱势，回避**

#### 板块六：资金面分析

**必须分析的数据：**

- 主力净流入/流出金额
- 超大单、大单、中单、小单分别的净流入
- 判断资金行为模式：
  - 超大单流入 + 小单流出 → 机构吸筹
  - 超大单流出 + 小单流入 → 机构出货
  - 全面流入 → 市场一致看好
  - 全面流出 → 恐慌性抛售

#### 板块七：筹码分析

**必须分析的数据：**

- 获利比例：100%=全员获利（无套牢盘），<50%=大量套牢
- 平均成本：与现价对比判断安全边际
- 筹码集中度（90%/70%）：越小越集中，主力控盘越强

#### 板块八：机构观点汇总

- 汇总最新5条机构研报的评级和核心观点
- 提取一致目标价
- 分析看多/看空的主要逻辑分歧

#### 板块九：风险评估矩阵

**必须评估的风险因子：**

| 风险因子 | 评估维度 |
|---------|---------|
| 短期回调风险 | RSI超买程度、涨幅过大 |
| 估值风险 | PE/PB历史分位、PEG水平 |
| 业绩不达预期 | 增速可持续性、下游需求变化 |
| 行业竞争 | 可比公司动态、市场份额变化 |
| 宏观/政策风险 | 汇率、利率、行业政策 |
| 流动性风险 | 成交量、换手率、股东集中度 |

**评分方法：**
- 概率：高(>60%) / 中(30-60%) / 低(<30%)
- 影响度：高 / 中 / 低
- 综合评分 = 概率 × 影响度

#### 板块十：投资策略

**必须覆盖的投资者类型：**

1. **已持仓-重仓** → 持有/减仓策略
2. **已持仓-轻仓** → 持有/加仓策略
3. **未持仓-积极型** → 建仓策略
4. **未持仓-稳健型** → 观望/建仓策略

**每种策略必须包含：**
- 具体操作区间（价格）
- 分批建仓比例（如 20%/30%/30%/20%）
- 目标价（短期/中期）
- 止损价（必须明确）
- 关键催化剂/时间节点

**敏感性分析表：**
```markdown
| 全年增速假设 | 对应净利润 | EPS | 合理PE | 合理股价 | vs 现价 |
|------------|----------|------|-------|---------|--------|
| +30% | ... | ... | ... | ... | ... |
| +40% | ... | ... | ... | ... | ... |
| +50% | ... | ... | ... | ... | ... |
| +60% | ... | ... | ... | ... | ... |
| +70% | ... | ... | ... | ... | ... |
```

### Phase 3：报告输出

**输出规范：**

1. **格式**：Markdown 文件
2. **文件命名**：`{股票名称}_{分析类型}_{日期YYYYMMDD}.md`
   - 示例：`沪电股份_2026Q1业绩预告深度分析_20260414.md`
3. **报告头部**：必须包含分析日期、当前股价、总市值
4. **报告结尾**：必须包含一句话核心结论 + 免责声明
5. **数据精度**：金额保留2位小数，百分比保留2位小数，PE/PB保留2位小数
6. **货币单位**：A股用人民币¥，标注"元"或"亿元"

**一句话结论模板：**
```
{股票名称}{事件}（+{增速}%），{核心逻辑}。中长期看¥{目标价}，但当前¥{现价}{风险判断}。最优策略是{操作建议}。
```

## 常见分析场景

### 场景1：业绩预告分析
用户说："XX发了业绩预告，帮我分析"
→ 重点关注：预告增速区间、是否超预期、隐含的季度环比趋势

### 场景2：业绩快报/年报分析
用户说："XX年报出了，深度分析一下"
→ 重点关注：全年趋势、Q4边际变化、ROE变动、资产扩张情况

### 场景3：投资策略生成
用户说："XX值不值得买？给个策略"
→ 重点关注：估值合理性、安全边际、建仓时机、止损设置

### 场景4：持仓决策
用户说："我持有XX，现在该怎么操作？"
→ 重点关注：技术面趋势、筹码成本、资金流向、减仓/加仓节点

## 涨跌颜色规范

遵循中国A股市场惯例：
- **红色 🔴** = 上涨/流入 = 利好
- **绿色 🟢** = 下跌/流出 = 利空
