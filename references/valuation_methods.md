# 估值方法论详解

## 一、PE 估值法（市盈率法）

### 原理
股价 = EPS × 合理PE倍数

### 适用范围
- ✅ 盈利稳定或高增长的公司
- ❌ 亏损公司、周期底部公司

### 合理PE的确定方法

**方法1：历史分位法**
- 取过去3-5年PE的中位数作为合理PE
- 若当前PE > 历史90%分位 → 高估
- 若当前PE < 历史10%分位 → 低估

**方法2：行业对比法**
- 取同行业可比公司PE均值
- 龙头公司可给予10-20%溢价

**方法3：增速匹配法**
- 高增速（>50%）：PE可给35-50x
- 中增速（20-50%）：PE可给20-35x
- 低增速（<20%）：PE可给15-25x
- 负增长：PE下修至10-15x

### 多年度PE计算

```
PE (2026E) = 当前市值 / 2026年预期净利润
PE (2027E) = 当前市值 / 2027年预期净利润
```

当两年维度的PE差异大时，说明市场在price-in未来增长。

## 二、PEG 估值法

### 原理
PEG = PE / 净利润增速(%)

### 评判标准（A股市场）

| PEG | 评价 | 说明 |
|-----|------|------|
| < 0.5 | **极具吸引力** | 成长远未被定价 |
| 0.5 - 0.8 | **较有吸引力** | 成长性价比高 |
| 0.8 - 1.0 | **合理** | 增速与估值匹配 |
| 1.0 - 1.5 | **偏贵** | 部分透支未来 |
| > 1.5 | **高估** | 过度乐观 |

### 使用注意
- 增速用**未来1-2年的预期增速**，不用历史增速
- 增速要可持续，一次性收益导致的高增速不算
- PEG更适合成长股，不适合价值股/周期股

### PEG=1 目标价计算
```
目标价 = 预期EPS × 预期增速(%)
```
示例：EPS=3.0元，增速=50% → 目标价 = 3.0 × 50 = ¥150

## 三、PS 估值法（市销率法）

### 原理
PS = 市值 / 营收

### 适用范围
- ✅ 高增长但未盈利的公司
- ✅ SaaS/互联网/生物科技等轻资产公司
- ❌ 低毛利率的传统制造业

### 评判参考
- PS < 1 → 可能被低估（需排除基本面恶化）
- PS = 2-5 → 一般公司的合理区间
- PS = 5-15 → 高增长公司
- PS > 15 → 需要极高增速支撑

---

## 四、PB 估值法（市净率法）

### 原理
PB = 市值 / 净资产

### 适用范围
- ✅ 银行、保险、地产等重资产行业
- ✅ 周期底部的制造业
- ❌ 轻资产公司（品牌/技术公司）

### 与ROE的关系
合理PB ≈ ROE / (折现率 - 增长率)

简化规则：
- ROE > 20% → PB可给3-6x
- ROE = 10-20% → PB可给1.5-3x
- ROE < 10% → PB应在1-1.5x
- ROE < 5% → PB可能低于1x（破净）

## 五、DCF 估值法（简化版）

### 适用范围
- ✅ 现金流稳定的成熟公司
- ❌ 高增长期、现金流不稳定的公司

### 简化计算（两阶段模型）

```
阶段1（高增长期，5年）：
自由现金流 = 当年净利润 × (1 - 资本开支率) × (1 + 增速)^n

阶段2（永续期）：
终值 = 第5年现金流 × (1 + 永续增速) / (折现率 - 永续增速)

合理市值 = Σ(阶段1各年现金流的现值) + 终值的现值
```

### 关键参数参考

| 参数 | 保守 | 中性 | 乐观 |
|------|------|------|------|
| 折现率(WACC) | 12% | 10% | 8% |
| 永续增速 | 2% | 3% | 4% |
| 资本开支率 | 40% | 30% | 20% |

## 六、横向对比法

### 方法
选取3-5家同行业可比公司，对比以下指标：

| 指标 | 目标公司 | 可比1 | 可比2 | 可比3 | 均值 |
|------|---------|------|------|------|------|
| PE (TTM) | | | | | |
| PE (Forward) | | | | | |
| PB | | | | | |
| ROE | | | | | |
| 营收增速 | | | | | |
| 净利增速 | | | | | |

若目标公司增速>均值但PE<均值 → 可能被低估

## 七、隐含增速法（逆向推导）

### 原理
从当前股价反推市场隐含的增速预期，再与实际/预期增速对比。

### 计算步骤

```
1. 当前PE = 当前市值 / TTM净利润
2. 假设"合理PE"（如行业中位数30x）
3. 隐含增速 = (当前PE / 合理PE - 1) × 100%
```

或更精确：
```
当前股价 = EPS(next year) × 合理PE × (1 + 隐含超额增速)
隐含超额增速 = 当前股价 / (EPS_next × 合理PE) - 1
```

### 判断标准
- 实际增速 > 隐含增速 → 股价仍有上行空间
- 实际增速 ≈ 隐含增速 → 股价合理
- 实际增速 < 隐含增速 → 股价可能高估

## 八、综合估值建议

### 推荐使用的估值组合

| 公司类型 | 主要方法 | 辅助方法 | 验证方法 |
|---------|---------|---------|---------|
| 高增长科技股 | PEG | PE(Forward) | 隐含增速法 |
| 稳定消费股 | PE(TTM) | DCF | 股息率 |
| 周期制造股 | PB | PE(周期中枢) | 横向对比 |
| 亏损成长股 | PS | EV/Revenue | 可比交易 |
| 金融股 | PB | PE | 股息率 |

### 合理价值区间的确定

1. 用3种以上方法分别计算目标价
2. 取中位数作为"合理价值中枢"
3. 最低值作为"保守目标"，最高值作为"乐观目标"
4. 最终呈现为区间：¥保守 - ¥乐观

```
合理价值区间 = [最低估值方法结果, 最高估值方法结果]
合理价值中枢 = 所有方法结果的中位数
```
